# Discord Bot

친목 서버를 위한 올인원 Discord 봇입니다.

포인트 경제 시스템을 중심으로 미니게임과 도박으로 경쟁하고, 파티를 모집하고, 음악을 틀고, AI와 대화하고, 리그 오브 레전드 전적까지 조회할 수 있습니다. 서버에서 자주 하는 것들을 하나의 봇으로 해결합니다.

---

## 기능

### 🪙 포인트 시스템
서버 내 자체 포인트 경제. 출석·게임·도박으로 포인트를 얻고 잃으며 경쟁합니다.

| 명령어 | 설명 |
|--------|------|
| `!등록` | 최초 1회 등록, 1,000P 지급 |
| `!정보` | 닉네임 · 포인트 · 가입일 확인 |
| `!출석` | 하루 1회, 기본 1,000P / 4% 확률 2,000P |
| `!게임기록` | 최근 5판 게임 전적 + 승률 |
| `!도박기록` | 최근 5판 도박 기록 + 수익률 |

---

### 🎮 미니게임
`!게임` 명령어로 드롭다운에서 선택. 승패에 따라 포인트가 변동됩니다.

| 게임 | 규칙 | 포인트 |
|------|------|--------|
| 참참참 | 봇과 방향이 다르면 승리 | 승 +100P / 패 -100P |
| 가위바위보 | 봇과 대결 | 승 +100P / 비 0P / 패 -100P |
| 제비뽑기 | 승 60% / 패 40% | 승 +100~500P / 패 -100~700P |

---

### 🎰 도박
`!도박` 명령어로 드롭다운에서 선택. 포인트를 직접 배팅합니다.

| 게임 | 규칙 | 배당 |
|------|------|------|
| 홀짝 | 1~100 주사위, 홀/짝 선택 | 승 +100% / 패 -100% |
| 경마 | 말 3마리 중 하나에 배팅 | 1등 +150% / 2등 -50% / 3등 -100% |

---

### 🎵 음악
유튜브 검색 후 드롭다운으로 선택 재생. 서버 음성 채널에서 사용합니다.

| 명령어 | 설명 |
|--------|------|
| `!music <검색어>` | 유튜브 검색 후 선택 재생 (별칭: `!노래`, `!음악`) |
| `!musiclist` | 대기열 확인 / `r` 옵션으로 곡 제거 |
| `!pause` | 일시정지 / 재개 토글 |
| `!skip` | 현재 곡 스킵 |

---

### 👥 파티 모집
시간을 지정해 파티를 만들면, 해당 시간에 참여자 전원에게 멘션 알림을 보냅니다.

| 명령어 | 설명 |
|--------|------|
| `!party <YYYY/MM/DD/HH/MM> <제목>` | 파티 생성 |
| `!partylist` | 파티 목록 |
| `!partydel <번호>` | 파티 삭제 |
| `!partymembers <번호>` | 파티 멤버 확인 |

---

### 🤖 AI 챗봇
Google Gemini 기반. 단발성 질문과 멀티턴 대화 모두 지원합니다.

| 명령어 | 설명 |
|--------|------|
| `!g <질문>` | 단발성 질문 (별칭: `!ai`) |
| `!c` | 멀티턴 챗봇 스레드 시작 (최대 10회, 별칭: `!chat`) |

---

### 🎮 리그 오브 레전드
Riot API 연동. 소환사 이름으로 프로필과 최근 전적을 조회합니다.
검색 또는 즐겨찾기로 소환사를 찾을 수 있으며, 즐겨찾기는 두 명령어가 공유합니다.

| 명령어 | 설명 |
|--------|------|
| `!롤프로필` | 랭크 · 마스터리 · 최근 전적 요약 |
| `!롤전적` | 솔로랭크 / 자유랭크 최근 5게임 상세 전적 |

전적에는 KDA · 킬관여율 · 딜량 · CS · 게임시간과 함께 포지션 보정 등급이 표시됩니다.

| 등급 | 승리 | 패배 |
|------|------|------|
| 최상 | 🔥 캐리 | 🔥 혼자함 |
| 상 | ✅ 활약 | ✅ 선방 |
| 중 | 😐 평범 | 😐 평범 |
| 하 | 💀 발목 | 🐀 트롤 |

## 실행

```bash
# 의존성 설치
pip install -r requirements.txt

# 환경 변수 설정 (.env)
token=<Discord Bot 토큰>
GEMINI_API_KEY=<Google Gemini API 키>
DATABASE_URL=postgresql+asyncpg://<user>@<host>:<port>/<db>
RIOT_API_KEY=<Riot API 키>  # developer.riotgames.com 에서 발급

# DB 마이그레이션
alembic upgrade head

# 실행
python -m bot.main
```
